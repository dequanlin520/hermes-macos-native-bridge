does not implement `HermesProtocolClient`, does not read Keychain or browser
data, does not submit a user prompt, and does not probe guessed endpoints.

allowlisted `hermes` executable and package install location. Tracked evidence
uses `<hermes-install>` instead of private absolute paths. Generated runtime
logs and source extracts stay under `artifacts/m2-003/`, which is ignored.

## Decision

M2-003 VERDICT: GO

Proceed only with a narrow protocol client whose initial surface is:

- fixed loopback base URL supplied by `HermesProcessSupervisor`;
- `GET /api/status` status/version/auth-mode discovery;
- authenticated JSON-RPC 2.0 over `/api/ws`;
- session creation/status, event dispatch, cooperative interrupt, and approval
  response models as source-defined RPC methods.

Do not implement generic REST probing, guessed endpoints, or active run
submission tests in this spike.
